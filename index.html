01 / Deep Learning

EEG Sleep Stage Classifier

DS 340 · Sep — Dec 2025
  • Built a deep learning model that classifies sleep stages from single-channel EEG using the Sleep-EDF dataset.
  • Extracted EEG channels from EDF files and re-segmented variable-length hypnogram intervals into 30-second epochs — produced ~2,600 labeled samples; corrected uncertain labels via forward-fill.
  • Designed a 2-layer LSTM with Batch Normalization and Dropout per layer for stability and generalization. Tuned window length, batch size, and learning rate.
  • Reached ~86% test accuracy; validated with 5-fold and time-series cross-validation. Confusion matrix showed strong performance on Wake and N2, with expected confusion at N1 boundaries.
  • Hypnogram visualization confirmed the model reproduced realistic sleep flow.

Steam Game Hit Predictor

BA 305 · Sep — Dec 2025
  • Built a machine learning pipeline on ~2,300 Steam bestsellers to predict whether a title would become a "hit," focusing on factors that structurally drive commercial success.
