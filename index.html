03 / Selected Work

01
— 01 / Deep Learning

EEG Sleep Stage Classifier

DS 340 · Sep — Dec 2025

Developed a deep learning model to automate sleep stage classification from single-channel EEG using the Sleep-EDF dataset. Extracted EEG channels from EDF files and re-segmented variable-length hypnogram intervals into 30-second epochs, producing ~2,600 labeled samples; uncertain labels were corrected via forward-fill to keep the dataset consistent. Each epoch was reshaped into a 3,000-step time series for LSTM input.

Designed a 2-layer LSTM with Batch Normalization and Dropout per layer for stability and generalization. Tuned window length, batch size, and learning rate, reaching ~86% test accuracy. Validated with 5-fold and time-series cross-validation; the confusion matrix showed strong performance on Wake and N2, with expected confusion at N1 boundaries. Hypnogram visualization confirmed the model reproduced realistic sleep flow.

— 02 / Machine Learning

Steam Game Hit Predictor

BA 305 · Sep — Dec 2025

Built a machine learning pipeline on roughly 2,300 Steam bestsellers to predict whether a title would become a "hit," focusing on which factors structurally drive commercial success. Designed feature engineering from scratch: removed leaky and uninformative variables, recoded text-based features as dummy variables, and applied log transforms and IQR-based outlier removal to numeric data to stabilize distributions.

After observing weak linear-model performance, expanded to Decision Tree, Random Forest, and Logistic Regression and compared models using accuracy, precision, and recall. Translated outputs into interpretable insights for both developers and players: content length and broad platform support showed positive impact, while in-app monetization correlated negatively. Results were framed as decision-ready guidance rather than raw metrics.

— 03 / Graph Analytics

Amazon Co-Purchase Graph

DS 210 · Rust Implementation

Analyzed consumer purchase patterns and product-to-product relationships in a graph of ~260,000 nodes and 1.2M+ edges built from Amazon co-purchase data. Implemented in Rust for memory safety and performance at scale — parsed the raw dataset into a directed graph and structured the loading and construction pipeline to handle the volume efficiently.

Computed degree distributions to map connection patterns, surfacing clusters where a small set of products are repeatedly co-purchased. Implemented Dijkstra's algorithm to compute shortest paths and derived closeness centrality, identifying the highest-influence nodes in the network. Translated the findings beyond raw metrics into directional input for recommendation system improvements and cross-sell strategy.
